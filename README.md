## Run locally

You need have Node.js 22 or higher installed.

```bash
  git clone git@github.com:zvolcsey/react-shop.git

  cd react-shop

  npm ci

  cd client

  npm ci

  cd ..

  cd server

  npm ci

  cd ..

  npm run dev # Running the client and the server
```
